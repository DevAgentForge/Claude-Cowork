# Welcome to Agent Cowork

Build domain-specialized AI assistants with **Agent Cowork** - an extensible desktop application powered by Claude and the Anthropic Agent SDK.

## What is Agent Cowork?

Agent Cowork is an Electron-based desktop application that enables you to create customized AI assistants for specific domains, industries, or workflows. Built with React 19, TypeScript, and Tailwind CSS, it provides a modern development experience with powerful customization options.

### Key Features

- **Desktop Application**: Native macOS, Windows, and Linux support
- **Persistent Sessions**: SQLite-backed conversation history
- **Tool Integration**: File system access, code execution, and search capabilities
- **Personality System**: Config-based AI specialization for different domains
- **MCP Support**: Integrate external tools and data sources
- **Extensible**: Hooks, custom agents, and build variants
- **Modern Stack**: React 19, TypeScript, Zustand, Tailwind CSS 4

## Who is This For?

### For Users

- **Medical Professionals**: HIPAA-compliant medical coding assistants
- **Legal Teams**: Document review and contract analysis tools

### For Developers

- **React Developers**: Familiar stack with modern patterns
